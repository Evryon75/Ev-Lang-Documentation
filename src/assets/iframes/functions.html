Variables

Functions in Ev are declared on top scope, the syntax does not mean much under the hood.
It could be said that the lowest scope is some sort of main function, as it is still a block.
The parameters are added as new variables on top scope.

Example:

        fun add = (x, y) {
            return x + y;
        };

If no return statement is provided it will return a null value, returned values can be ignored and that is the purpose of
procedures since they return null. Attempting to create a null variable using this and calling it will result in an error

Example:

        fun output_add = (x, y) {
            output(x + y);
        };

Functions can be declared inside other functions in Ev

Example:

        fun one = (x) {
            fun two = (y) {
                
            };
        };
