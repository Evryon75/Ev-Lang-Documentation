Introduction to the Ev Programming Language

The Ev Programming Language is a general purpose language.
It is dynamically typed and relatively high level.
The interpreter is made in rust ðŸ¦€, it goes through tokenizing, parsing, and walking the abstract syntax tree.

Here is an example program:

Source code:

    let x = "Hello,";
    output(x, "world!");

Output:

    [SYS] Lexing finished successfully
    [SYS] Parsing finished successfully
    [OUT] Hello, world!
    [SYS] Process finished successfully
    Press [ENTER] to close, or input "DEBUG" to access debug information
    Both "DEBUG" and "debug" will work

[SYS] and [OUT] are aligned in the program, this is just a font related error

To execute your source code simply put the interpreter in a folder together with a file named "main.ev",
if no such file is found the interpreter will run into an error and immidiately close.

Visual Studio Code is a great IDE to work in Ev, you can edit main.ev and simply run this command 
    start ev_interpreter.exe
(or whatever the interpreter's name is if you choose to rename it) in your terminal.
Alternatively you can also just double click the interpreter from your file explorer.
Dont forget you can press the up arrow key to cycle through your previous terminal commands!
