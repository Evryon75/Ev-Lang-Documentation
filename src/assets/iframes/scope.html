Scopes in Ev

When entering a new block, a new scope layer is added.
Declaring a function or a variable will save it in the current top scope. When exiting a block the top scope is removed.

There is a built in function [scopes()] that outputs the scope from where it is used, here is an example

Source code:

        let x = 1;
        fun s = () {
            let y = 2;
            scopes();
        };
        s();
