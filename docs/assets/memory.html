Memory

Memory in Ev is represented by rust vectors, there are 2: one for variables and one for functions.
The vectors hold hashmaps, with the identifier name as the key and the expression or function type, containing the parameters and the block
as the key. 

    let mut variables: Vec<HashMap<String, Expression>> = vec![];
    let mut functions: Vec<HashMap<String, Function>> = vec![];

You can free a variable or function before its scope ends with a built in function.
This is useful to recursion

Example:

    fun rec = () {
        let temp = "a";
        free(temp);
        rec();
    };

If we didn't do this we would run into an error stating that the variable "temp" already exists in this scope, since we never exit it in this case.
