Input and Output

We can take input with the built in function,
we need to pass a string parameter that specifies the type of input we are trying to get,
"str" for strings and "num" for numbers

Example for strings:

    let string_var = input("str");

Example for numbers:

    let number_var = input("num");

If we try to input a string for a number input an error will occur

Example:

    [SYS] Lexing finished successfully
    [SYS] Parsing finished successfully
    > not a number
    [ERR] Walking error: you cannot convert letters to a number 
    This includes empty spaces
    Press [ENTER] to close

To output something, simply use the built in function and pass as many parameters are you want.
They will be printed with a space in between them

Example:

    output("Hello", "world!);

Output:

    [SYS] Lexing finished successfully
    [SYS] Parsing finished successfully
    [OUT] Hello world!
    [SYS] Process finished successfully
    Press [ENTER] to close, or input "DEBUG" to access debug information
    Both "DEBUG" and "debug" will work

Note: "output" makes more sense than "print" to me
