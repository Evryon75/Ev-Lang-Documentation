Aborting a process

You may need to stop the process from running at any point, and display an error message.
There is a built in function just for that.

Example:

    if 1 == input("num") {
        abort("You cannot input 1");
    };

Output:

    [SYS] Lexing finished successfully
    [SYS] Parsing finished successfully
    > 1
    Process stopped: "You cannot input 1" 
    Press [ENTER] to close

It can work without any parameters too.
