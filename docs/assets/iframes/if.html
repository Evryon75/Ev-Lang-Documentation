If Statements

If statements in Ev require a semicolon at the end of the section,
the conditional expression does not need to be grouped in parentheses,
and there is no else-if statement, an additional if statement can be used in the else statement

Example:

    if 1 == 1 {
        // Block
    } else {
        // Else block
        if 2 == 2 {
            // Block
        };
    };
